Special functions

  1. Closure function (Function within another function is called as closure function. Outer function cannot access the properties of inner function where as inner function access the properties of outer function. )
  2. Function chaining (It is nothing but multiple functions are called on the same object consecutively.)
  3. Call back function (passing function as an input argument to another function is known as call back function).
