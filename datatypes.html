In Js we have 2 types of data types

  1. primitive data types (also called as immutable)
  2. non-primitive data types(also called as mutable)

1.primitive data types :-

  1. Numbers (All numeric values comes under numbers)
    eg. 1,-1,0,0.1,2/3-----
  2. String (All alphabetical values / any values enclosed within double/single quotes comes under string.
    eg. name = "kalyani" )
  3. Boolean (Boolean values takes either true/false)
    eg. var isMarried=true
  4. Null
    eg. var name=null
  5. Undefined (when we declare variable without initializing will comes under undefined. )
    eg. var name;
  6. Symbol (symbols in Js are unique in nature.no symbol is equal to another).
    eg. var mySym = symbol('hello') var mySym1 = symbol('hello')
    if(mysym == mysym1){ console.log(true)}. though both are same we get false
  7. Bigint ( We use this to store large numbers. To use bigint we need to include 'n' to the number)
    eg. var num = 46789865n

2.Non primitive data types

  1. Array ( It is a special variable which can hold more than one value at a time).
    eg. var array =['a','b']

    we have 2 types of arrays

    1. Homogeneous array (collection of elements which are of same type).
      eg. var arr = ['a','b','c']
    2. Heterogeneous array (collection of elements which are not same data type).
      eg. var arr = ['a',10,true,null,{}]

    2. (we can write array in 2 ways)
    eg. 1 : var arr = ['a','b','c']
    eg.2 : var fruit = new Array('apple','banana')

    Array which contains multiple objects is known as complex array.
    eg. var det=[{name:'kal',sal:50000},{name:'bav',sal:100000}]

  2. Object (It is a entity which has some properties & prop are in the form of key & value pairs).
    eg. object{key : value}

  3. Date (It is used to get the dates).

  5. Function (function is a block of code designed to perform particular task.we need to call the function to use it.)

    Types of functions

    1. Named functions ( function which contains name)
    2. Anonymous func (func without name.In order to call this func we need to assign this func to variable. & call the func with this var)
    3. Function expression (The process of assigning func to var is called as functional expression)
